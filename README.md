# 🔐 Password Generator

A simple and secure Password Generator built using HTML, CSS, and JavaScript to create strong and random passwords.

---

## Features
- Generate random passwords instantly
- Customize password length
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Copy password to clipboard
- Responsive and user-friendly interface

---

##  Tech Stack
- HTML5
- CSS3
- JavaScript

---
